Configurar Firebase

  1. Ve a js/firebase-config.example.js y copia tu configuración de Firebase.
  2. Crea Firestore (modo de producción) y Storage en tu proyecto de Firebase.
  3. Habilita Authentication → Email/Password
  4. Sube el sitio a GitHub Pages.
